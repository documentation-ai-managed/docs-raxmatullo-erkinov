## Document Organization and Folders

Organize your documentation using intuitive folder structures. You create top-level folders for major categories like "API Reference" or "User Guides," then nest subfolders for finer granularity.

<Steps>
  <Step title="Create a Folder" icon="folder-plus">
    Navigate to the root directory. Click the new folder icon and name it `guides`.
  </Step>
  <Step title="Nest Subfolders" icon="folder">
    Inside `guides`, create `quickstart` and `advanced`. Drag documents between folders.
  </Step>
  <Step title="Reorder" icon="move">
    Use drag-and-drop to rearrange. Folders sync across all team members.
  </Step>
</Steps>

<Callout kind="tip">
  Prefix folder names with numbers like `01-introduction` for automatic sorting.
</Callout>

## Collaboration and Sharing Options

Collaborate by inviting team members with role-based permissions. You share public links or embed documents.

<CodeGroup tabs="Invite,Share">
  ```javascript
  // API to invite user (for integrations)
  ```
