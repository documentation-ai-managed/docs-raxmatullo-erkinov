<Update label="2024-10-15" description="v1.3.0" tags={["feature", "improvement"]}>

## New Features

- Added support for custom brand colors in documentation themes, using your `{brandColor}` like `#3B82F6`.
- Introduced `<Columns>` component for responsive grid layouts in MDX pages.

## Improvements

- Enhanced search functionality with fuzzy matching for better result accuracy.
- Optimized page load times by `>30%` through lazy-loading components.

## Bug Fixes

- Fixed parsing errors with nested code blocks using 4 backticks.
- Resolved issues with special character escaping in prose text.

</Update>

<Update label="2024-09-25" description="v1.2.0" tags={["feature", "bugfix"]}>

## New Features

- New `<Video>` and `<Iframe>` components for embedding media and external content.
- Added `ParamField` and `ResponseField` for structured API documentation.

## Bug Fixes

- Corrected heading hierarchy enforcement, preventing H1 usage in body content.
- Fixed attribute parsing for JSX components, ensuring double quotes for strings.

## Improvements

- Updated Mermaid diagram rendering to use safe, simple syntax only.

</Update>

<Update label="2024-08-10" description="v1.1.0" tags={["feature", "breaking"]}>

## New Features

- Full support for `<Update>` components in changelog pages with tag arrays.
- Integrated Lucide icons across all components like `<Card icon="rocket">`.

## Breaking Changes

- Removed legacy `<Mermaid>` JSX component; use fenced code blocks with `mermaid` language tag instead.

## Bug Fixes

- Patched self-closing component errors for containers like `<Steps>`.

</Update>
